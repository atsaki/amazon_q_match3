# Amazon Q Match3

Amazon Q Developerを使って作成したPygameベースのMatch3パズルゲームです。

## 🎮 ゲーム概要

- **ジャンル**: パズルゲーム（マッチ3系）
- **制限時間**: 30秒/1分/3分から選択可能
- **目標**: 同じ色のブロックを3つ以上揃えて高スコアを目指す

## 🎯 ゲームルール

1. 隣接する2つのブロックをクリックして交換
2. 同じ色のブロックを縦または横に3つ以上揃える
3. マッチしたブロックが消えて得点獲得
4. 上からブロックが落下して新しいマッチが発生する可能性
5. 連鎖でボーナス得点
6. 制限時間内でできるだけ高スコアを狙う

## 🎨 ゲーム要素

- **8×8のゲームグリッド**
- **6色のカラフルなブロック**（赤、青、緑、黄、紫、オレンジ）
- **スコアシステム**: 3個=100点、4個=200点、5個=500点
- **連鎖システム**: 連続消去で追加得点
- **制限時間選択**: 30秒/1分/3分の3モード

## ✨ 新機能

### 🕐 制限時間選択
- **30秒モード**: 短時間集中型
- **1分モード**: バランス型
- **3分モード**: じっくり戦略型

### 🏆 ハイスコアシステム
- 各モード別にトップ10を記録
- プレイヤー名、スコア、日時を保存
- 全モード通しての最高スコア表示
- ゲーム中にベストスコア表示

### 🔄 ゲーム再開機能
- ゲーム終了後に同じモードで再プレイ
- メインメニューに戻って別モード選択
- ESCキーでいつでもメニューに戻る

### 📊 拡張UI
- 残り時間の色分け表示（緑→黄→赤）
- 現在のモード表示
- リアルタイムベストスコア表示
- 直感的なメニューシステム

## 🚀 セットアップ

### 前提条件
- Python 3.8以上
- uv（パッケージマネージャー）

### インストール

```bash
# リポジトリをクローン
git clone <repository-url>
cd q_example

# 依存関係をインストール

```

## 🎮 ゲーム実行

```bash
# 全機能付きゲームを起動

```

すべての機能が統合されています：
- 制限時間選択（30秒/1分/3分）
- ハイスコア記録・表示
- ゲーム再開機能

- 包括的ログ機能
- エラーハンドリング

### メニュー操作
- **マウスクリック**: メニュー項目の選択
